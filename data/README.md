# Data

Raw rasters and processed CSVs are generated by the notebook and are excluded from the repository (see `.gitignore`). Run the notebook to reproduce them.

## GEE Collections

```python
import ee
ee.Authenticate()
ee.Initialize(project='YOUR_GEE_PROJECT_ID')

# MODIS burned area
ee.ImageCollection('MODIS/061/MCD64A1')

# MODIS land cover
ee.ImageCollection('MODIS/061/MCD12Q1')

# Sentinel-5P CO
ee.ImageCollection('COPERNICUS/S5P/OFFL/L3_CO')

# Sentinel-5P NO2
ee.ImageCollection('COPERNICUS/S5P/OFFL/L3_NO2')

# Sentinel-5P Aerosol Index
ee.ImageCollection('COPERNICUS/S5P/OFFL/L3_AER_AI')

# Hansen Global Forest Change v1.11
ee.Image('UMD/hansen/global_forest_change_2023_v1_11')

# Country boundaries
ee.FeatureCollection('USDOS/LSIB_SIMPLE/2017')
```
